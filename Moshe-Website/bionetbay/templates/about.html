BioNetBay: Repository Overview


BioNetBay is an online repository that stores gene sets and gene networks.

You can browse the repository and dowload files by visiting the Resources tab.

You can contribute a file to the repository by visiting the the Contribute tab.

Please note, that you must be signed in in order to contribute to the repository. Click on the Login tab to sign in or to create an account.

How the repository works

The repository consists of three basic components: Resource, Dataset and File

Resource:

Resource refers to the source of the data, either as the original generator or curator of the data.
Examples: Allen Brain Altas, BioGRID
Resources contian Datasets.


Dataset:

Dataset refers to the specific set of data from the resource.
Examples: Allen Brain Altas-Adult Human Brain, Allen Brain Altas-Adult Mouse Brain
Datasets contian Files.


File:

Files make up the heart of the repository.
Files contain gene association/interaction libraries in the form of either a GMT file or a SIG file.

How the repository is categorized

Datasets are categorized as one of three possible categories based on gene information: Interaction, Association or Ontology

Interaction:

Interaction refers to any gene-gene set and has the following subcategories:
Kinase-Substate, Protein-Protein and Transcription Factor-Target


Association:

Association refers to any set of gene-non gene set and has the following subcategories:
Tissue, Disease and a more generic Sample


Onotology:

Ontology refers to any gene set based on an ontology and has the following subcategories:
Biological Process, Cellular Component and Molecular Function
